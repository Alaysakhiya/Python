## 📋 Description

The **Interactive Personal Data Collector** is a beginner-friendly Python program designed to demonstrate fundamental programming concepts including:

- User input handling
- Data type conversion and validation
- Variable memory management
- Basic arithmetic operations
- Command-line interface design

This project is perfect for beginners learning Python as it covers essential concepts in a practical, interactive way.

---

## ✨ Features

### Core Features

- ✅ **Interactive User Input**
  - Collects name, age, height, favorite number, and current year from users
  - Provides clear and intuitive prompts for better user experience

- ✅ **Data Type Handling**
  - Supports multiple data types: strings, integers, and floating-point numbers
  - Demonstrates type conversion in Python

- ✅ **Memory Analysis**
  - Displays memory addresses of stored variables using Python's built-in `id()` function
  - Helps beginners understand how Python manages memory

- ✅ **Birth Year Calculator**
  - Automatically calculates approximate birth year based on current year and age
  - Demonstrates basic arithmetic operations

- ✅ **Cross-Platform Compatibility**
  - Works on Windows, Linux, and macOS
  - No external dependencies required

### Additional Features

- ✅ **Clean Code Structure**
  - Well-commented code for easy understanding
  - Follows Python best practices and coding standards

- ✅ **Error Handling**
  - Basic input validation for data types
  - User-friendly error messages

- ✅ **Extensible Design**
  - Easy to add new features or modify existing ones
  - Modular code structure

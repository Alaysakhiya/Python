## ✨ Features

- ✅ Collects user data through command-line interface
- ✅ Automatically handles different data types (str, int, float)
- ✅ Displays memory addresses of stored variables
- ✅ Computes approximate birth year based on age
- ✅ Clean and user-friendly prompts

---

## 🛠️ Prerequisites

- **Python 3.6** or higher
- A text editor (VS Code, PyCharm, etc.)
- Command-line terminal

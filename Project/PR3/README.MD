## 🚀 Overview

**Student Data Organizer** is a menu-driven Python application designed to manage student records efficiently through an interactive command-line interface.

The project provides essential record management operations such as adding, viewing, updating, and deleting student information while demonstrating practical use of Python data structures.

---

## ✨ Features

### 👨‍🎓 Student Management

* ➕ Add New Student Records
* 👀 View Student Information
* ✏️ Update Existing Records
* ❌ Delete Student Records
* 📚 Display Student Subjects
* 🆔 Automatic Student ID Generation

### 📋 Student Information

Each record contains:

* Name
* Age
* Grade
* Date of Birth
* Subjects

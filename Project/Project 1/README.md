## 📖 Overview

Interactive Personal Data Collector is a simple Python console application that collects personal information from users and demonstrates fundamental Python concepts.

This project is ideal for beginners learning:

- User Input
- Variables
- Data Types
- Type Casting
- Memory Addresses
- Arithmetic Operations

---

## ✨ Features

- 📝 Collect user information
- 🔢 Accept integer and float inputs
- 🏷️ Display variable data types
- 🧠 Show memory addresses using `id()`
- 🎂 Calculate approximate birth year
- 💻 Interactive command-line interface

---

## 📋 Information Collected

The program asks the user for:

- Name
- Age
- Height (in feet)
- Favourite Number
- Current Year

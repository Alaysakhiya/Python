## 🚀 Overview

Data Analyzer & Transformer is a menu-driven Python application that enables users to perform common data analysis operations on numerical datasets.

The project demonstrates practical implementation of Python programming concepts including recursion, lambda expressions, list manipulation, sorting techniques, and statistical computations.

---

## ✨ Features

### 📥 Data Input

* Accept multiple numerical values from the user
* Store data dynamically using Python lists

### 📊 Data Summary

Generate a quick overview of the dataset:

* Total Elements
* Minimum Value
* Maximum Value
* Sum of Values
* Average Value

### 🔁 Recursive Factorial Calculator

* Calculate factorials using recursion
* Demonstrates recursive problem-solving techniques

### 🎯 Threshold Filtering

* Filter data based on user-defined conditions
* Implemented using Lambda Functions

### 🔢 Sorting Operations

* Ascending Order
* Descending Order

### 📈 Dataset Statistics

Display important statistical information instantly.

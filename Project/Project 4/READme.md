## 🚀 Overview

Data Analyzer & Transformer is a menu-driven Python application that allows users to perform various operations on numerical datasets, including analysis, filtering, sorting, and statistical calculations.

This project demonstrates practical implementation of Python fundamentals such as **functions**, **recursion**, **lambda expressions**, and **data manipulation techniques**.

---

## ✨ Features

* 📥 Input and store numerical datasets
* 📊 Generate data summaries
* 🔁 Calculate factorial using recursion
* 🎯 Filter data using threshold values
* 🔢 Sort data in ascending or descending order
* 📈 Display dataset statistics

---

## 🛠️ Tech Stack

* Python 3
* Functions
* Recursion
* Lambda Functions
* Lists & Built-in Methods

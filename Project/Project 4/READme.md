## 🚀 Overview

Data Analyzer & Transformer is a menu-driven Python application that enables users to perform various data processing operations on numerical datasets.

The project showcases practical implementation of:

✔ Functions

✔ Recursion

✔ Lambda Expressions

✔ Sorting Techniques

✔ Statistical Analysis

✔ User Interaction

---
